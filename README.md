# Ecommerce-RTO-Dashboard
### **Overview**
Interactive Excel dashboard to analyze Ecommerce Return-to-Origin trends
### **Dashboard Preview**
!RTO Dashboard](Picture 1.png)
###**Key Business Insights***
- **Total Revenue Tracked**:₹76.23Lakhs
- **Overall RTO%**:3.60%
- **Problem Category**:Fashion leads with 5.58% RTO
- **Top Revenue Cities**:Bangalore,Kolkata,Mumbai
- **Payment Mode Analysis**:COD vs Prepaid RTO comparison included  
###**Features**
- Revenue by Category & City Charts
- RTO% by Category & Payment Mode
- YOY Revenue Growth trendline
- Dynamic Slicers for filtering
###**Tools and Skills Used**
- Microsoft Excel
- Pivot Tables & Pivot Charts
- Data Cleaning & Visualization
- Dashboard Design & Business Analysis
